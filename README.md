cordova-anguar-frontend-sample
==============================

This is hybrid app based on angular.js and is packaged by cordova.
This app can be tested with [oauth2-restapi-server](https://github.com/vinebrancho/oauth2-restapi-server)

Used cordova plugin
- InAppBrowser
- Console
